# Sistema de Farmácia

Sistema desktop de controle de estoque e vendas para farmácia, desenvolvido em Java com arquitetura MVC.

## Funcionalidades

- Cadastro de medicamentos (nome, quantidade, tipo de embalagem)
- Controle de estoque
- Registro de vendas
- Interface gráfica com Swing

## Arquitetura

O projeto segue o padrão MVC:

```
src/
└── Dobackaofront/
    ├── model/       # Entidades e regras de negócio
    ├── view/        # Interface gráfica (Swing)
    └── controller/  # Lógica de controle
```

## Como rodar

Abra o projeto no IntelliJ IDEA ou Eclipse e execute a classe principal.

**Requisitos:** Java 17+
